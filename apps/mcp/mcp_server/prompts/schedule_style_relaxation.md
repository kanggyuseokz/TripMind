# 휴양형 여행 일정 생성 가이드

## 여행 스타일 특징
- **목표**: 여유롭고 편안한 휴식
- **페이스**: 느린 일정, 충분한 여유 시간
- **우선순위**: 휴식 > 관광 > 식사

## 일정 구성 원칙

### 시간 배분
- **오전 (09:00-12:00)**: 가벼운 활동 또는 호텔 휴식
- **점심 (12:00-13:00)**: 여유로운 식사 (1시간)
- **오후 (14:00-15:30)**: 선택적 관광 또는 자유시간
- **카페 (15:30-17:00)**: 뷰 좋은 카페에서 휴식
- **저녁 (18:00-19:00)**: 편안한 저녁 식사
- **밤 (19:00-)**: 호텔 복귀, 자유 시간

### 활동 선택 기준 (Google Maps 평점)
1. **스파/마사지**: 평점 4.5+ / 리뷰 200+
2. **해변/공원**: 평점 4.3+ / 조용한 분위기
3. **카페/라운지**: 평점 4.6+ / 뷰 좋은 곳
4. **쇼핑**: 평점 4.0+ / 여유롭게 둘러보기
5. **관광지**: 평점 4.2+ / 1일 1~2곳 이하

### 이동 최소화
- 호텔 근처 활동 우선
- 이동 시간 30분 이내
- 대중교통보다 택시/호텔 셔틀 추천

## 날짜별 템플릿

### 첫날
```
09:00 - 호텔 체크인 및 짐 정리
12:00 - 호텔 근처에서 가벼운 점심
14:00 - 호텔 시설 둘러보기 (수영장, 스파 등)
18:00 - 호텔 레스토랑 또는 근처에서 저녁
21:00 - 호텔에서 휴식
```

### 중간일
```
09:00 - 늦은 아침 식사 (브런치)
11:00 - 선택적 활동 (스파, 해변, 쇼핑)
13:00 - 여유로운 점심
15:00 - 자유 시간 (호텔 휴식 또는 가벼운 산책)
18:00 - 저녁 식사
21:00 - 호텔 복귀
```

### 마지막날
```
09:00 - 느긋한 아침 식사
11:00 - 체크아웃 전 마지막 호텔 시설 이용
12:00 - 가벼운 점심
14:00 - 공항 이동 (여유있게)
```

## 주의사항
- ❌ 빡빡한 일정 금지
- ❌ 이른 아침 출발 지양
- ❌ 하루 3곳 이상 방문 금지
- ✅ 매일 2시간 이상 자유시간 확보
- ✅ 점심 시간 충분히 배정
- ✅ 호텔/숙소 중심 일정
