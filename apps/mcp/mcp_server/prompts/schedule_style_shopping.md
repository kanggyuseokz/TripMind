# 쇼핑형 여행 일정 생성 가이드

## 여행 스타일 특징
- **목표**: 쇼핑 및 현지 제품 구매
- **페이스**: 여유롭되 전략적인 일정
- **우선순위**: 쇼핑 > 식사 > 가벼운 관광

## 일정 구성 원칙

### 시간 배분
- **오전 (10:00-13:00)**: 백화점/쇼핑몰 (개장 시간 고려)
- **점심 (13:00-14:30)**: 쇼핑 지역 내 식사
- **오후 (14:30-19:00)**: 추가 쇼핑 (아울렛, 시장)
- **저녁 (19:00-21:00)**: 저녁 식사 및 야간 쇼핑
- **밤 (21:00-)**: 면세점 또는 24시간 쇼핑몰

### 쇼핑 장소 우선순위
1. **면세점**: 공항 또는 시내 면세점
2. **백화점/명품 거리**: 럭셔리 브랜드
3. **아울렛**: 할인 쇼핑
4. **전통 시장**: 기념품, 현지 제품
5. **쇼핑 거리**: 트렌디한 패션/잡화

### 쇼핑 최적화
- 짐 보관 서비스 활용
- 면세 한도 확인
- 환급 가능 매장 우선
- 세일 기간 확인

## 쇼핑 체크리스트
- [ ] 여권 사본 (면세 구매용)
- [ ] 신용카드 해외 사용 승인
- [ ] 쇼핑 리스트 작성
- [ ] 예산 배분
- [ ] 추가 캐리어/가방 준비
- [ ] 택스리펀 앱 설치

## 예산 배분 가이드
```
명품/럭셔리: 40%
의류/패션: 25%
화장품/향수: 15%
기념품/잡화: 10%
식품/특산물: 10%
```

## 주의사항
- ✅ 면세 한도 확인 (귀국 시)
- ✅ 환급 서류 꼼꼼히 작성
- ✅ 가격 비교 (온라인 vs 오프라인)
- ✅ 진품 확인 (명품 구매 시)
- ❌ 충동 구매 자제
- ❌ 무게 제한 초과 주의
- ❌ 액체류 기내 반입 규정 확인
