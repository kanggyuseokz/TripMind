# 액티비티형 여행 일정 생성 가이드

## 여행 스타일 특징
- **목표**: 액티브한 체험 중심
- **페이스**: 에너지 넘치는 일정
- **우선순위**: 액티비티 > 관광 > 식사

## 일정 구성 원칙

### 시간 배분
- **오전 (07:00-12:00)**: 메인 액티비티 (체력 필요)
- **점심 (12:00-13:00)**: 빠른 식사 및 휴식 (1시간)
- **오후 (13:00-15:00)**: 추가 액티비티 또는 가벼운 활동
- **카페 (15:00-16:00)**: 회복 시간 (에너지 충전)
- **저녁 전 (16:00-18:00)**: 자유 시간 또는 마무리 활동
- **저녁 (18:00-19:00)**: 저녁 식사 및 회복
- **밤 (20:00-)**: 야간 액티비티 또는 휴식

### 활동 선택 기준 (Google Maps 평점)
1. **수상 스포츠**: 평점 4.5+ / 안전성 확인
2. **산악 활동**: 평점 4.4+ / 난이도 체크
3. **익스트림 스포츠**: 평점 4.3+ / 리뷰 필수 확인
4. **사이클링**: 평점 4.2+ / 코스 정보
5. **카페**: 평점 4.5+ / 에너지 보충 메뉴

### 체력 관리
- 고강도 액티비티는 1일 1개
- 전날 과격한 활동 후 다음날 가볍게
- 충분한 수분 섭취 시간
- 점심 후 회복 시간 필수

```
14:00 - 공항 이동
```

## 주의사항
- ✅ 사전 예약 필수 (인기 액티비티)
- ✅ 날씨 확인 (우천 시 대체 계획)
- ✅ 보험 가입 확인
- ✅ 체력 수준 고려
- ❌ 연속 고강도 활동 금지
- ❌ 부상 위험 높은 활동 신중히
- ❌ 과도한 음주 금지
