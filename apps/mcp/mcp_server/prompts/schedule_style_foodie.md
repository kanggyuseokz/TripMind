# 맛집형 여행 일정 생성 가이드

## 여행 스타일 특징
- **목표**: 현지 음식 경험 최대화
- **페이스**: 식사 중심 일정
- **우선순위**: 맛집 > 카페/디저트 > 시장

## 일정 구성 원칙

### 시간 배분
- **아침 (08:00-09:00)**: 현지 아침 식사 명소 (1시간)
- **브런치 (10:30-11:30)**: 카페/베이커리 (1시간)
- **점심 (12:30-13:30)**: 대표 맛집 방문 (1시간)
- **간식 (15:00-16:00)**: 인기 카페/디저트
- **저녁 (18:30-19:30)**: 파인다이닝/현지 맛집 (1시간)
- **야식 (21:00-22:00)**: 야시장/포장마차

### 활동 선택 기준 (Google Maps 평점 기반)
1. **미슐랭/유명 맛집**: 평점 4.5+ / 리뷰 1000+
2. **현지인 맛집**: 평점 4.3+ / 리뷰 500+ / 현지어 리뷰 많음
3. **길거리 음식**: 평점 4.2+ / 활기찬 분위기
4. **카페/디저트**: 평점 4.6+ / 시그니처 메뉴
5. **전통 시장**: 평점 4.0+ / 다양한 먹거리
6. **베이커리**: 평점 4.5+ / 갓 구운 빵
7. **야시장**: 평점 4.2+ / 야간 분위기

### 식사 간 활동
- 소화 시간 확보 (산책 30분)
- 식재료 시장 방문
- 가벼운 관광 (사진 촬영)
- 카페에서 휴식

## Google Maps 맛집 선택 팁
- ✅ "미슐랭 가이드" 라벨 확인
- ✅ 현지어 리뷰 비율 높을수록 진짜 맛집
- ✅ 사진 필터: 음식 사진만 확인
- ✅ 혼잡한 시간대 피하기 (대기 시간 확인)
- ✅ "예약 필요" 태그 확인
- ❌ 관광객 리뷰만 많은 곳 주의

## 카페/디저트 추천 기준
- 평점 4.6 이상
- 시그니처 메뉴 명확
- SNS에서 유명 (인스타그래머블)
- 현지 특산 재료 사용
- 갓 만든 신선한 제품

## 예약 필수 체크리스트
- [ ] 미슐랭 레스토랑 예약 (1~2개월 전)
- [ ] 인기 맛집 영업시간/웨이팅 확인
- [ ] 휴무일 확인 (월요일 많음)
- [ ] 1끼당 예산 책정
- [ ] 식이 제한/알레르기 미리 전달

## 주의사항
- ✅ 식사 예산 충분히 확보
- ✅ 식사 시간 1~1.5시간 이내
- ✅ 소화 시간 30분 이상 확보
- ✅ 평점 4.3 이상 우선
- ❌ 연속 과식 지양 (간격 2시간+)
- ❌ 같은 음식 중복 피하기
- ❌ 평점 4.0 미만 맛집 배제
- ❌ 관광객 전용 식당 주의
