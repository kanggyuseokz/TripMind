# 관광형 여행 일정 생성 가이드

## 여행 스타일 특징
- **목표**: 주요 관광지 최대한 방문
- **페이스**: 활동적이고 체계적인 일정
- **우선순위**: 관광 > 카페/휴식 > 식사

## 일정 구성 원칙

### 시간 배분
- **오전 (08:30-12:00)**: 주요 관광지 방문 (2곳)
- **점심 (12:00-13:00)**: 효율적인 식사 (1시간)
- **오후 (13:00-15:00)**: 추가 관광지 방문 (1~2곳)
- **카페 (15:00-16:00)**: 인기 카페에서 휴식
- **저녁 전 (16:00-18:00)**: 마지막 관광 또는 쇼핑
- **저녁 (18:00-19:00)**: 저녁 식사
- **야간 (19:00-21:00)**: 야경/야시장
- **복귀 (21:00-)**: 호텔 복귀

### 활동 선택 기준 (Google Maps 평점 기반)
1. **주요 관광지**: 평점 4.3+ / 리뷰 1000+
2. **박물관/미술관**: 평점 4.2+ / 문화시설
3. **역사 유적지**: 평점 4.0+ / 역사적 가치
4. **전망대/랜드마크**: 평점 4.4+ / 포토존
5. **현지 시장**: 평점 4.0+ / 활기찬 분위기
6. **카페**: 평점 4.5+ / 리뷰 200+ / 인스타그래머블
7. **레스토랑**: 평점 4.3+ / 리뷰 500+ / 현지 음식

### 이동 최적화
- 지역별 묶어서 방문
- 대중교통 적극 활용
- 이동 시간 최소화 (도보 15분 이내 우선)
- Google Maps로 실시간 이동시간 확인

## Google Maps 활용 팁
- ✅ "인기 있는" 라벨 확인
- ✅ 현재 혼잡도 체크
- ✅ 영업 시간 확인 (휴무일 주의)
- ✅ 사진/리뷰 미리 확인
- ✅ 근처 추천 장소 연계 방문

## 카페 추천 기준
- 평점 4.5 이상
- 인스타그래머블한 인테리어
- 시그니처 메뉴 있음
- 와이파이/충전 가능
- 앉을 자리 충분
- 조용한 분위기 (휴식용)

## 주의사항
- ✅ 개장 시간 확인 필수 (월요일 휴무 많음)
- ✅ 동선 효율적으로 구성
- ✅ 1일 5~7곳 방문 적정 (카페 포함)
- ✅ 이동 시간 현실적으로 배정 (도보 15분 이내)
- ✅ 카페 휴식으로 체력 회복
- ❌ 식사 시간 2시간 이상 금지
- ❌ 너무 먼 거리 이동 지양 (30분 이내)
- ❌ 평점 4.0 미만 장소 배제
